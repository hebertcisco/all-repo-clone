# Getting started

## Installation

### PowerShell

```pwsh
iwr https://unpkg.com/all-repo-clone/lib/install.ps1 -useb | iex
```

### NPM

```bash
npm i -g all-repo-clone
```

## Usage
```bash
all-repo-clone -u hebertcisco
# or
all-repo-clone --username hebertcisco
```
